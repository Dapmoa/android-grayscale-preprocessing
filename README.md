# Android Grayscale Preprocessing

Repository ini berisi script Python untuk mengubah file APK Android menjadi citra grayscale sebagai tahap prapemrosesan dataset machine learning.

## Isi Repository

- convertmal.py
- convertbegn.py
- Manual Book (txt)

## Cara Menjalankan

### Malware

```bash
python3 convertmal.py
```

### Benign

```bash
python3 convertbegn.py
```
